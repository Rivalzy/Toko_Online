## How To Run

1. Clone this repository
```
https://github.com/Rivalzy/Toko_Online.git
cd toko_onlie
```
2. Set up the environment
```
## Install dependencies
composer install

## Migration MYSQL
php artisan migrate

## Seeding data
php artisan db:seed

## Generate key
php artisan key:generate
```
3. Run the server
```
php artisan serve
```
Server will run on port http://127.0.0.1:8000
